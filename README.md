# Мой первый проект

**Цель проекта:**  
Освоить основы создания веб-страниц и работы с системами контроля версий.

**Стек технологий:**  
- HTML
- CSS
- JavaScript

**Структура проекта:**  
- `index.html` — основная страница  
- `css/` — стили  
- `js/` — скрипты  
- `images/` — изображения

**Как запустить:**  
Откройте `index.html` в браузере.
